# Vyken AI Posture - AI Governance Gap Checker

A self-contained interactive tool to assess your organization's AI governance posture through 10 targeted questions.

## 🚀 Quick Start

Simply open `index.html` in any modern web browser. No installation or build process required.

## 📋 Features

- **10 Critical Questions** - Assess your AI governance across policy, enforcement, visibility, and compliance
- **Instant Results** - Get a scored assessment with specific gap identification
- **No Dependencies** - Completely self-contained HTML file with inline styles and scripts
- **Mobile Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **No Data Collection** - All processing happens in the browser, no data is sent anywhere

## 🎨 Design

- Modern dark theme with cyan accents
- Smooth animations and transitions
- Professional typography using Syne and DM Sans fonts
- Responsive grid layout

## 🔧 Technical Details

- **Zero Dependencies** - No npm packages, no build tools, no frameworks
- **Pure HTML/CSS/JavaScript** - Works in any modern browser
- **Fonts** - Loaded from Google Fonts CDN (Syne & DM Sans)
- **File Size** - Single ~40KB HTML file

## 📱 Browser Support

Works in all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers
